# ATRIUM — лендинг архитектурного бюро

Посадочная страница архитектурного бюро на чистом HTML, CSS и JavaScript — без фреймворков и сторонних библиотек. Проект выполнен в рамках практического задания по дисциплине «Тенденции проектирования графических пользовательских интерфейсов».

**Автор:** Буряков Артём Иванович · ТКМО-02-25 · 2026

---

## Целевая аудитория

### Primary — владельцы недвижимости, 28–45 лет

Планируют строительство или реновацию жилья. Изучают сразу несколько бюро параллельно, решают медленно, доверяют визуальному контенту и социальным доказательствам больше, чем рекламным обещаниям. Около 60% первичных просмотров приходят с мобильных устройств.

### Secondary — девелоперы и владельцы бизнеса, 35–55 лет

Управляющие компании, владельцы коммерческой недвижимости, заказчики общественных пространств. Ценят портфолио схожих объектов, репутацию бюро и список крупных клиентов.

### Боли пользователей

- **Страх недобросовестных подрядчиков** — непрозрачная работа, срывы сроков, скрытые доплаты
- **Непонимание реальной стоимости и сроков** архитектурного проектирования
- **Сложная коммуникация** — язык архитекторов непонятен заказчику без технического образования

## Продуктовая гипотеза

> Блок портфолио с прозрачными ценами «от» в сочетании с социальным доказательством (логотипы клиентов, отзывы со звёздами) повышает конверсию в заявку до 3–5% — против 1–2% у бюро без указания порядка цен.

Гипотеза опирается на анализ 6 ведущих архитектурных бюро (Snøhetta, RAUM, Atelier, ASADOV, SIO, ABG), выполненный в предыдущих практических работах. Лендинг должен закрывать ключевые возражения пользователя ещё до контакта с менеджером.

### Метрики успеха

| Метрика | Целевое значение |
|---|---|
| Конверсия в заполнение формы | 3–5% |
| Глубина скролла до формы | ≥ 60% посетителей |
| Кликабельность hero-CTA | ≥ 15% |

## UX-обоснование

### Визуальный язык

Холодная нейтральная палитра + один акцентный цвет — типичный приём премиум-бюро (Snøhetta, Studio Gang, SANAA). Архитектура не должна спорить с фотографиями архитектуры.

| Роль | Цвет | HEX |
|---|---|---|
| Фон | Белый | `#FFFFFF` |
| Основной текст | Тёмно-серый | `#1C1C1E` |
| Вторичный текст | Серый | `#6B7280` |
| Акцент (CTA, italic) | Синий | `#1E5FA8` |
| Тёмные секции | Глубокий тёмный | `#0F1923` |

### Типографика

- **Cormorant Garamond** для заголовков — serif с характером, передаёт премиальность и уважение к традиции
- **Inter** для основного текста — нейтральный sans-serif с хорошей читаемостью на всех размерах
- **Italic-акценты синим** в заголовках выделяют ключевое слово: «архитектура, которая *остаётся* в памяти места»

### Композиционные решения

- **Нумерация секций** (01 · 02 · 03 · 04) — приём из премиум-бюро, помогает ориентироваться в длинной странице
- **Тонкие линии-разделители** вместо жёстких границ между секциями
- **Много воздуха** между блоками — приучает глаз к спокойному ритму
- **Hover-эффекты** на карточках портфолио — синий оверлей с кратким описанием проекта, плавный transition 0.4s
- **Параллакс Hero** — фон движется медленнее страницы, создавая ощущение глубины при скролле

### Mobile-first

Около 60% первичных просмотров — с мобильных. Поэтому верстка строится от мобильного к десктопу через три брейкпоинта:

| Брейкпоинт | Что меняется |
|---|---|
| **≤ 480 px** | Одна колонка, бургер-меню, hero центрируется, кнопки во всю ширину |
| **≤ 768 px** | Сетка портфолио 2 колонки, контакты под формой |
| **≥ 1024 px** | Полная сетка 3 колонки, sticky-header, hero-параллакс |

## Реализованный функционал

### Структура страницы

1. **Header** — sticky-навигация с логотипом, ссылками на разделы, бургер-меню на мобильных
2. **Hero** — полноэкранный блок с параллакс-фоном, заголовком-манифестом и двумя CTA
3. **О бюро** — философия + фото студии + счётчики достижений (16 лет, 80 проектов, 18 специалистов)
4. **Портфолио** — сетка 3×2 проектов с категорией, годом, ценой «от» и hover-оверлеем
5. **Социальное доказательство** — 6 логотипов клиентов + 3 отзыва со звёздами и аватарами
6. **CTA-секция** — тёмный блок с urgency-сообщением и пульсирующей точкой
7. **Контакты** — форма заявки + информация о студии
8. **Footer** — логотип, разделы, услуги, иконки соцсетей (VK, Telegram, Behance)

### JavaScript-функционал

- **IntersectionObserver** — плавное появление блоков (`fade-up` + `translateY`) с каскадной задержкой при попадании в viewport
- **Параллакс Hero** — фон движется медленнее страницы через `transform: translate3d` на событие `scroll`
- **Sticky-header** — белый фон с `backdrop-filter: blur(16px)` появляется после 80 пикселей скролла вниз
- **Бургер-меню** — toggle класса `.open`, блокировка скролла body при открытом меню
- **Автоматическое форматирование телефона** — пользователь вводит `9151811827`, поле автоматически превращает это в `+7 (915) 181-18-27`. Цифра `8` в начале заменяется на `7`, начальный `+7` сохраняется, международные номера с другим кодом страны не трогаются
- **Валидация формы** — email и телефон проверяются на `blur`, поле с ошибкой подсвечивается красной рамкой. При попытке отправить незаполненную форму все пустые поля подсвечиваются одновременно

### CSS-решения

- CSS-переменные для палитры и шрифтов — упрощают изменение темы
- `scroll-behavior: smooth` + якорные ссылки для плавной навигации
- `clamp()` для адаптивных размеров заголовков
- CSS Grid + Flexbox для сеток
- `transition` с `cubic-bezier(0.4, 0, 0.2, 1)` для естественных анимаций

## Технологический стек

- **HTML5** — семантическая разметка (`<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`)
- **CSS3** — CSS-переменные, Grid, Flexbox, медиазапросы, transitions
- **JavaScript** (vanilla) — IntersectionObserver, обработчики событий, валидация
- **Google Fonts** — Cormorant Garamond, Inter
- **Unsplash** — фотоконтент

Никаких зависимостей. Никаких сборщиков. Никаких фреймворков.
